Key Bindings Example

Enter key bindings strings below, one per line, for testing, then press "Update". Some things to note. Handlers that don't use the Control, Alt, or Meta keys will not work when an input, textarea, or other editable element is focused. This is by-design to avoid clobbering native text manipulation commands. The bindings in the colored boxes will only work if the corresponding inputs have focus. This allows for context- sensitive bindings.
